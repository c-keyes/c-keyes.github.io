Reading: Start Module 2

Exercises:

Some SQL questions:
  • How can we access information from a different table while querying (e.g. while searching for batting statistics, display player's name instead of ID)?
  • Can we find the most common last name of MLB players over the years? Is SQL the right tool for this?
  • What could we measure and analyze if it was tracked by statcast?
Hitting metrics:
  • Impact on runs scored: BA accounts for about 67%, defensive outs accounts for less than 1% (this makes sense!)
  • Somewhat surprisingly, home runs allowed accounts for about 24% of variation in runs scored. A possible explanation could be that more home runs hit accross the league leads to more home runs hit by each time, and thus more runs scored in those seasons that everyone hit more home runs. Another could be playing in a home run friendly park
  • Whenever we are asking how one metric is related to another, we must consider the other factors at play, especially if we want to compare across historical eras.
  • Understand how wOBA is calculated. How might we approximate this if we didn't have massive amounts of historical base-out data?

Week 4 (2/15/22)

Reading: Continuing with Module 2

Exercises:

  • With a runner on first and no outs, how confident should you be to send the runner to second in a stolen base attempt? This can be determined by finding the change in run expectancy of the relevant base-out states and seeing what level of confidence gives a positive value (it's around 70% for 0, 1, and 2 outs!).
SQL comments:
  • The aggregate functions (AVG, SUM, STDDEV, etc.) allow us to aggregate information.
  • The GROUP BY modifier allows us to aggregate over subsets. For example, we might calculate average home runs for each team since 2010.
  • The HAVING command allows conditionals just like we would use WHERE on column data; for example we might only display those teams averaging more than some number of home runs per year.
  • This could also be useful for computing career totals, including grouping by position for fielding data, etc.
